# Markup keys on a keyboard in Markdown

`<kbd>foo</kbd>` can be used to markup keys on a keyboard correctly. <kbd>foo</kbd> will be key-like.

```markdown
Press <kbd>Ctrl</kbd> + <kbd>D</kbd> to scroll down.
```

Press <kbd>Ctrl</kbd> + <kbd>D</kbd> to scroll down.
